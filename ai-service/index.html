WHAT WE DO

Claudeで、こんなことができます

大がかりなシステム刷新ではなく、
「いまの業務を楽にする」ところから始められる3つの領域です。

  • 業務自動化・効率化

    メール仕分け、議事録の要約、書類からのデータ抽出、定型レポートの自動生成。Claude Code／Agent SDKを組み合わせて、人が判断するべきところだけ人に残す設計を行います。

    使っている既存ツール（スプレッドシート／Notion／基幹システム等）にAIをつなぐ、カスタムスキルやMCPの実装。現場のワークフローを崩さず、そこにAIを"足す"形でご支援します。

ABOUT AI USAGE FEES

プラン料金とは別に、
Claudeのサブスクリプション費用が発生します。

月額定額だから予算が読める。合わなければいつでも解約OKです。

プラン料金 ≠ AI料金

上記プランは「設計・開発・導入・初期サポート」の費用です。実際にAIを動かすためのClaudeサブスクリプション費用は、Anthropic社から直接お客様へご請求となります（プラン料金とは別途発生）。

月額定額・いつでも解約OK

Claudeは月額のサブスクリプションプランで運用します。毎月の費用が固定なので予算化しやすく、使った分だけ上下する従量課金とは違って経費の予測がつきます。合わないと感じたら月末でいつでも解約可能。契約期間の縛りはありません。

どのプランから始める？

まずはClaude Proからスタートがおすすめです。月額$22（税込・約3千円台）で主要な業務を動かせます。実運用のなかで「もっと使いたい」となった段階で、Claude Max（月$110〜／税込）にアップグレードすれば一気にキャパが広がります。ダウングレードも自由なので、事業フェーズに合わせて調整できます。

従量課金API運用もご相談ください

基本はサブスクプランでのご提案ですが、大量バッチ処理や社内ツール埋め込みなど、使い方によっては従量課金の Claude API の方が向くケースもあります。オーバースペックを避け、貴社の用途に一番合った形をご相談時にご提案します。

FAQ

よくあるご質問

契約前に気になりやすいポイントをまとめました。

  • Q1 相談から導入開始まで、どれくらいかかりますか？

    標準的な流れは以下です。

    1. 無料相談（30分〜1時間）：即日〜3営業日で日程調整
    2. お見積もり・提案書：相談から3〜5営業日以内にご提示
    3. ご契約・NDA締結：1〜2営業日
    4. 導入開始

    全体で、初回相談から導入着手まで 約1〜2週間 が目安です。

  • Q2 機密情報を扱っても大丈夫ですか？

    ご契約前にNDA（秘密保持契約）を締結可能です。開発時の社内データは、作業完了後に速やかに削除します。Claude APIはAnthropic社の規約により、入力データがモデルの学習に使われない仕様となっています。

    ただし、個人情報・医療情報・決済情報など特に機密性の高いデータを扱う場合は、セキュリティ要件を事前にすり合わせさせてください。

  • Q3 対応できる業種・会社規模の目安は？

    中小企業（従業員数〜300名程度）を中心に、バックオフィス・カスタマーサポート・営業・士業・小売・製造業の管理部門など、幅広く対応しています。規模の大きなご依頼はパートナーと連携してお受けするケースもあります。

  • Q4 Claude Opus以外のモデルは使えますか？

    はい。初期設計はOpusを基本としていますが、用途に応じて Sonnet・Haiku への切り替え、また業務ごとに使い分けるハイブリッド構成も可能です。精度・速度・コストのバランスでご提案します。

  • Q5 個人事業主だと、途中で引き継げなくなりませんか？

    その懸念は正当です。だからこそ当サービスでは、以下の方針で構築しています。

    • Claude API / Agent SDK / MCP など、標準技術のみで実装
    • プロンプトや設定ファイルはすべてお客様の管理下に納品
    • コード・設計ドキュメントは、第三者でも引き継げる形で整備

    将来的に他ベンダー・社内エンジニア・お客様ご自身への移管にも対応できる設計です。

  • Q6 料金に含まれないもの（スコープ外）は？

    以下は別途費用または対象外となります。

    • Claudeサブスクリプション費用（Pro / Max）お客様直払い ※AI料金について
    • 外部SaaSの有料プラン費用（Slack / Notion / Zapier 等）
    • サーバー・ドメイン・SSLなどのインフラ費用
    • 大規模UI/UX設計、ネイティブアプリ開発
    • 医療・金融・法務領域の専門監修

    見積もり時にスコープ表をお渡しするので、線引きはクリアです。

  • Q7 解約・途中終了はできますか？

    はい。制作フェーズは納品物ごとの精算、月額サポート契約は当月末での解約が可能です（違約金なし）。ただし制作中の成果物については、着手済み分の費用をご精算ください。

    Claudeサブスクリプション（Claude Pro / Max）はお客様名義でご契約いただくため、当方との契約解除後もそのままお使いいただけます。
